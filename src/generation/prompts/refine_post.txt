STEP 1: IDENTIFY LOAD-BEARING ELEMENTS (PROTECTED)
- The hook's structural mechanic (sentence count, rhythm, function per sentence)
- The core thesis (the ONE arrival)
- The anchoring scene (specific moment where founder is inside the experience)
- 1-2 voice signatures (rhythm/phrasing)
These stay. All edits orbit them.

STEP 2: TRIAGE FEEDBACK (STRICT BUCKETING)
FIX (max 3): Structural breaks, voice mismatches, logical holes, flat endings, stacked aphorisms, body drift from hook's promise.
CONSIDER (max 2): Low-cost clarity tweaks. Ignore if it alters register.
IGNORE: "Tone it down", "make it accessible", contradictory advice, generic CTAs, caveats.

STEP 3: STRUCTURAL AUDIT BEFORE EDITING

Run these checks from the framework:

HOOK-PROMISE CHECK: Restate the hook's promise in one sentence. Does every paragraph in the body advance THIS specific argument? Flag any drift to adjacent topics.

ONE-ARRIVAL CHECK: Count standalone quotable lines that could be screenshotted independently. If more than one, keep the sharpest (usually the last), restructure the others.

VISIBLE-MOMENT CHECK: Is there at least one scene you can picture? Is the founder inside it (not observing)? Does vulnerability beat authority?

RECYCLING CHECK: Could any paragraph be copy-pasted from one of the founder's existing published posts? If yes, flag for rewrite.

WORD-COUNT CHECK: Is the post within the founder's typical range? If over, cut.

STEP 4: SURGICAL EDITS (15-30% CHANGE MAX)
1. Word swaps > sentence rewrites > targeted cuts > structural moves.
2. Add proof, not padding. If feedback says "not specific", inject exact numbers/names.
3. Cut weakest paragraph entirely if pacing drags.
4. Rewrite last 1-2 lines if ending is flat. End on concrete detail. No summaries.

STEP 5: ANTI-GENERIC GATE
[ ] Post still sounds like {personality_card}, not a consultant
[ ] At least one rough edge/unpolished rhythm remains
[ ] Zero banned words or phrases added
[ ] Thesis is sharper, not safer
[ ] Change % is visibly 15-30%
[ ] Only ONE arrival — no stacked aphorisms
[ ] Body delivers on hook's exact promise — no drift
[ ] At least one scene where founder is vulnerable/exposed
[ ] Ending leaves a picture, not a lesson
