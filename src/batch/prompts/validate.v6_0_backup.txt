## 5-DIMENSION SCORING (0-5 each)

### 1. VOICE-MARKER COMPLIANCE
How many of the founder's voice markers appear naturally?
- 0 = none present
- 1 = 1 marker, forced or shallow
- 2 = 1 marker, well-integrated
- 3 = 2 markers, naturally expressed
- 4 = 3 markers, naturally expressed
- 5 = 4+ markers, seamlessly integrated

### 2. OPENER-RHYTHM MATCH
Does the opening follow the founder's documented argument rhythm pattern?
- 0 = completely different rhythm (corporate announcement vs. paradoxical declaration)
- 3 = partially matches (right shape, wrong register)
- 5 = matches the documented rhythm exactly

### 3. FORMATTING-HABIT MATCH
Does the post match the founder's formatting habits AND expected marker rates?
Check: em-dashes, smileys, hashtags, paragraph length, sentence structure, closing moves.
Compare against the EXPECTED MARKER RATES — posts that strip or force markers score lower.
- 0 = completely different formatting
- 3 = some formatting matches, rates partially aligned
- 5 = formatting indistinguishable from founder's published posts

### 4. REGISTER CHECK
Does the post read like the FOUNDER wrote it, or like someone else?
- 0 = reads like corporate copy, press release, or marketing
- 1 = reads like a generic thought leader
- 3 = reads like a founder in this space, but not THIS specific founder
- 5 = reads like the calibration paragraph's register — same voice, same posture

### 5. POSTURE CHECK (operator inside vs. commentator outside)
Is the founder positioned as an OPERATOR inside the experience, or a COMMENTATOR watching from outside?
- 0 = pure commentator — analyzes a category or trend as outsider
- 2 = mixed — some first-person but mostly third-person market commentary
- 3 = operator with distance — first-person but generic, not specific to this founder
- 5 = fully inside — speaks from lived operational experience

---

## DECISION RULE

- ANY dimension below 3 → return "FAIL"
- register_score below 3 → ALWAYS "FAIL" (most important signal)
- posture_score below 3 → "FAIL" (operator posture is non-negotiable)
