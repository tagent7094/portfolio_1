## 7-GATE DIAGNOSIS

Run the gates in order. Each gate is cheap to run on a draft that already passed the previous one.

### Gate 1: MECHANIC (does the opener stop the scroll?)
Identify which of the 13 proven mechanics is being used (see list below).
FAIL: Belief statements ("X matters"), question openers, "excited to announce," hedged setups, "I've been thinking about...", "What's your X strategy?"

### Gate 2: PLAUSIBILITY (would a real person actually say this in real life?)
Run on every opener containing a quote, scene, or stated dialogue. Failure shapes:
- **Strawman quote**: Character delivers a too-convenient line that exists only to be knocked down.
- **Counted-precision tic**: "She has been in eleven AI strategy meetings. Eleven." Real people don't count meetings.
- **Convenient setup**: Someone brags about something then can't answer the follow-up. Exists for the punchline.
- **Obvious-question from wrong person**: Board members ask about pipeline, not "when does AI handle everything?"
- **Impossible scenario**: Setup is logically impossible given the post.

If quote/scene with another person: would a real person in that role actually say this? If "no, too neatly set up" — FAIL.

### Gate 3: COHERENCE (does the opener match the post's actual argument?)
**Promise test**: Write what the opener promises in one sentence. Does every body paragraph deliver that promise?
**Compression test**: Compress the body to one sentence. Same argument as opener, or two different arguments?
Failure modes: drift (body wandered), bait-and-switch (opener sharper than body), two-posts-in-one.

### Gate 4: SPECIFICITY (no invented facts)
- First-degree ("Anant told me on the flight to Manila…") — ONLY if documented.
- Second-degree ("A founder I back…") — only if plausible at founder's scale.
- Third-degree ("Every founder I know who…") — always safe.
Default to third-degree when in doubt.

### Gate 5: MODE PRESERVATION
- **Declaring**: strongest opener = clean assertion, contrarian, specific number. Body defends straight.
- **Wrestling**: opener signals movement — confession, contradiction, stated tension.
Don't silently strip wrestling structure to make an opener punchier. The wrestling IS the point.

### Gate 6: VOICE FIT (does this opener sound like THIS founder?)
Two sub-tests:
1. **Recognition**: Would this founder read it and think "this is what I was going to say"? If it could appear in any executive's post — FAIL.
2. **Marker test**: Does the opener reflect at least 1 founder voice marker? If corporate register instead of founder's register — FAIL.

### Gate 7: SOURCE MIRROR FIDELITY (Batch A only)
Skip entirely if source_dissection is "N/A" (Batch B posts).

Three independent sub-checks. ANY failing = whole gate FAIL:

**Sub-check A — OPENER MECHANIC**: Does the opener use the SAME primary mechanic as the source? Same sentence count? Word count within ±20%?

**Sub-check B — BODY FORMAT**: Source body format must match generated body format exactly.
- numbered_list ↔ numbered_list (not flowing prose)
- bullet_list ↔ bullet_list
- parallel_paragraphs ↔ parallel_paragraphs
- (and so on for all 8 formats)

**Sub-check C — CLOSER MECHANIC**: Source closer mechanic must match generated closer mechanic.
- cta ↔ cta (not reframe_question)
- terminal_verdict ↔ terminal_verdict
- (and so on)

Do NOT give a charitable pass. The entire point of Batch A is mirroring.

## BODY SCAN

Read the body line by line. For each line ask:
- Could this sentence work as the opener?
- Does it have a number, juxtaposition, confession, quote, or scene?
- Is it punchier than line 1?
If yes, mark as buried gold.

---

## THE 13 PROVEN MECHANICS

Each variant must use a different one. Pick the 5 best fits for this post:
1. Specific number + context — lead with a precise figure from the draft.
2. Juxtaposition — two things that shouldn't coexist in the same sentence.
3. First-person confession — admit something with real cost.
4. Quote hook + reaction — someone said something, then your reaction.
5. Scene-entry — start in the middle of a physical moment.
6. Decision reveal — the outcome of a hard choice, stated plainly.
7. Contrarian declaration — state something most would disagree with (earned).
8. Pattern observation — notice something others miss.
9. Tension pair — two forces pulling in opposite directions.
10. Before/after snapshot — two states separated by time.
11. Micro story — a complete narrative arc in 1-2 sentences.
12. Authority redirect — cite external authority, then redirect their point.
13. Data inversion — commonly cited stat shown to mean the opposite.

## GENERATE 5 VARIANTS (A-E)

Rules per variant:
- Must use a DIFFERENT mechanic from the 13 above
- Must pass all 7 gates (including Gate 7 for Batch A)
- Must sound like this specific founder (use voice markers)
- No AI slop ("In today's...", "Here's the thing", "Hot take:", etc.)
- Each version must argue from a genuinely different angle — 5 variations of one idea ≠ 5 options
- If buried gold exists, use it as Variant A

High-lift elements (add 1-2, NOT all): specific number FROM draft, first-person verb, juxtaposition, emotion word, named entity FROM draft only.
