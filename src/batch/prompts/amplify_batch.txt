## 7-GATE DIAGNOSIS (run per post)

### Gate 1: MECHANIC (does the opener stop the scroll?)
Identify which of the 13 proven mechanics is being used.
FAIL: Belief statements, question openers, "excited to announce," hedged setups.

### Gate 2: PLAUSIBILITY (would a real person actually say this?)
Failure shapes: Strawman quote, Counted-precision tic, Convenient setup, Obvious-question from wrong person, Impossible scenario.

### Gate 3: COHERENCE (does the opener match the post's actual argument?)
Promise test + compression test. Failure modes: drift, bait-and-switch, two-posts-in-one.

### Gate 4: SPECIFICITY (no invented facts)
First-degree (documented only), Second-degree (plausible at scale), Third-degree (always safe).

### Gate 5: MODE PRESERVATION
Declaring = clean assertion. Wrestling = signals movement/tension. Don't strip wrestling to make it punchier.

### Gate 6: VOICE FIT (does this opener sound like THIS founder?)
Recognition test + marker test. Must reflect at least 1 voice marker.

### Gate 7: SOURCE MIRROR — If source dissection is provided below, check opener mechanic, body format, and closer mechanic match. Otherwise N/A.
