You are running the same-pack convergence test. This catches the failure mode where 9 posts each pass coherence individually but the pack as a whole reads as 9 ways to say the same thing.

The framework's argument matrix is supposed to prevent this; it doesn't always. The amplifier provides the second check.

## TASK

1. Read all compressed arguments above.
2. Group any that argue substantively the same thing — same thesis even with different examples, different phrasing, or different scenes.
3. Apply the convergence test:
   - **FAIL if 50% or more** of the compressed sentences argue substantively the same thing.
   - **FAIL if any two posts** use the same opening mechanic AND the same argument.
   - **PASS otherwise** — the pack has meaningful argument diversity.

4. If FAIL: identify which posts should be pushed back for rework. Ask for posts that argue meaningfully different takes. Don't keep amplifying versions that all converge on the founder's central thesis.

5. If FAIL: also produce a `replacement_angles` list of 3-5 DISTINCT, MUTUALLY EXCLUSIVE angles on the source material — each angle must be different enough that a post written to it could not collapse into the others. These are fed directly to the regen step, one per overlapping post.
